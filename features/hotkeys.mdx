# Hotkeys

Control Echophrase with customizable keyboard shortcuts.

## Default Hotkeys

| Action | Default Shortcut |
|--------|------------------|
| **Push-to-talk** | `Ctrl+Shift+Space` |
| **Toggle recording** | `Ctrl+Shift+R` |
| **Cancel recording** | `Escape` |

## Customizing Hotkeys

1. Open Echophrase settings
2. Navigate to **Hotkeys**
3. Click on any shortcut to change it
4. Press your desired key combination

## Tips

- Choose shortcuts that don't conflict with your other applications
- Push-to-talk is recommended for most users
- Toggle mode is useful for longer dictation sessions
